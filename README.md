# iSpaRo Taxonomy Explorer

Interactive explorer for 154 iSpaRo papers (2024-2025), mapped onto a robotics-paper phylogeny style taxonomy.

Live page:

https://gisbi-kim.github.io/isparo-taxonomy-explorer/

## What is included

- Technical vs strict taxonomy views
- Donut, tree, and curated-topic browsing modes
- Korean / English UI toggle
- Natural Korean abstract drafts
- Sentence-level abstract role labels, such as background, problem, gap, method, result, and limitation

## Files

- `index.html`: self-contained static explorer for GitHub Pages
- `isparo_taxonomy_mindmap.html`: same explorer under the development filename
- `isparo_taxonomy_assignment.csv`: paper-to-taxonomy assignment table
- `isparo_taxonomy_summary.md`: taxonomy summary

## Data Note

The paper metadata and citations are derived from the RoboPaper Atlas snapshot used during the build. Citation counts are a snapshot, not live values.
